Git Add Process

Easy to remember

  1. echo "# my-first-website" >> README.md
  2. Git init
  3. git add README.md
  4. git commit -m "first commit"
  5. git branch -M master
  6. git remote add origin https://github.com/rummanis/my-first-website.git
  7. git push -u origin master
  8. Use git pull to get data from github
  9. git checkout -b [branch name] Create a new branch and switch to it

If you wnat to pull data from git use git pull
